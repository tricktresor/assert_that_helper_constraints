# assert_that_helper_constraints

Helper Constraints for ASSERT_THAT.

## Überblick

Dieses Repository enthält Helper Constraints für `ASSERT_THAT` in ABAP.

## Sprache

- ABAP

## Zweck

Die enthaltenen Komponenten unterstützen das Formulieren und Wiederverwendbaren von Constraints in Tests mit `ASSERT_THAT`.

## Verwendung

Die genaue Verwendung hängt von den bereitgestellten ABAP-Objekten in diesem Repository ab. Typischerweise werden die Helper Constraints in ABAP-Unit-Tests eingebunden, um Erwartungen lesbarer und wiederverwendbar zu formulieren.
